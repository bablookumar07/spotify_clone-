# 🎵 Spotify Clone

A responsive Spotify-inspired web application built using **HTML5** and **CSS3**. This project was created to practice frontend development concepts, improve UI design skills, and replicate the look and feel of a real-world music streaming platform.

## ✨ Features

* Modern Spotify-inspired user interface
* Responsive design for different screen sizes
* Sidebar navigation menu
* Music cards and playlists section
* Sticky navigation bar
* Interactive hover effects
* Clean and organized code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Responsive Web Design
